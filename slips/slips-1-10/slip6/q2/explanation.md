# Slip 6 – Q2: JSP Implicit Objects

## What the program does
A JSP page that demonstrates the three key JSP implicit objects:
1. **`request`** – reads client IP, HTTP method, URI, User-Agent header
2. **`session`** – stores and retrieves a username attribute
3. **`response`** – adds a custom header
4. **`out`** – writes HTML directly from a scriptlet

## Java / JSP Concepts Used
- **JSP Implicit Objects** – `request`, `response`, `session`, `out`
- **JSP Scriptlet** `<% ... %>` – Java code inside JSP
- **JSP Expression** `<%= ... %>` – outputs value to page
- **HttpSession** – session management
- **HttpServletRequest** – reading client request details
