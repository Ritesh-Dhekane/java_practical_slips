# Slip 15 – Q2: Types of Inheritance

## What the program does
Demonstrates all three class-based inheritance types in Java:
1. **Single** – `Dog extends Animal`
2. **Multilevel** – `Child extends Parent extends GrandParent`
3. **Hierarchical** – `Car extends Vehicle` and `Bike extends Vehicle`

Also notes that **Multiple Inheritance** via classes is not supported in Java.

## Java Concepts Used
- **Inheritance** – `extends`
- **Single / Multilevel / Hierarchical** inheritance types
- **Method access** via inherited reference
