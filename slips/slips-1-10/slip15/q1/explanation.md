# Slip 15 – Q1: Array Reverse Copy

## What the program does
Stores 5 integers in an array. Copies all elements into a **new array in reverse
order** using index arithmetic `original[n-1-i]`. Displays both arrays.

## Java Concepts Used
- **Array** – fixed-size collection
- **Reverse indexing** – `reversed[i] = original[n-1-i]`
- **for loop** – iteration
