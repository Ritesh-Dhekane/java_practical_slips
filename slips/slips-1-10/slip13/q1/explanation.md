# Slip 13 – Q1: ArrayList – Names to Uppercase

## What the program does
Stores 5 student names (in lowercase) in an `ArrayList`. Iterates through
the list and converts each name to uppercase using `String.toUpperCase()`.
Displays both the original and transformed lists.

## Java Concepts Used
- **ArrayList** – dynamic list of strings
- **String.toUpperCase()** – converts string to uppercase
- **Enhanced for loop** – iterates over list
