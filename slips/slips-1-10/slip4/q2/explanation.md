# Slip 4 – Q2: Interface Shape – Circle & Rectangle

## What the program does
Defines an **interface** `Shape` with abstract method `area()`. The classes
`Circle` and `Rectangle` implement this interface and provide their own
`area()` calculation. Demonstrates polymorphism via a `Shape` array.

## Java Concepts Used
- **Interface** – `interface Shape` with abstract method
- **implements** – Circle and Rectangle implement Shape
- **Abstraction** – caller only needs to know `area()` contract
- **Polymorphism** – Shape references point to Circle/Rectangle objects
- **default method** – `displayArea()` in interface

## Syllabus Unit
**UNIT 1** – Interface, Abstraction, Polymorphism
