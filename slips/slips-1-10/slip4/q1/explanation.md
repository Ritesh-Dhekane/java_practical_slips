# Slip 4 – Q1: Second Highest Salary

## What the program does
Stores 5 salaries in an `ArrayList<Double>`, sorts them in ascending order,
then retrieves the **second highest** value using index `size - 2`.

## Java Concepts Used
- **ArrayList** – dynamic array
- **Collections.sort()** – ascending sort
- **Index-based access** – get element at `size - 2`

## Syllabus Unit
**UNIT 2** – Collections, ArrayList
