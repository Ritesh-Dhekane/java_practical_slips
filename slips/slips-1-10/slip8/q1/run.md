# How to Run – Slip 8 Q1 (JSP)

## Prerequisites
- Apache Tomcat 9.x

## Project Structure
```
MultiplyApp/
├── index.html
├── multiply.jsp
└── WEB-INF/
    └── web.xml
```

## Deploy & Run
1. Copy `MultiplyApp/` to Tomcat `webapps/`
2. Start Tomcat
3. Open: `http://localhost:8080/MultiplyApp/index.html`
4. Enter two numbers and click Calculate
