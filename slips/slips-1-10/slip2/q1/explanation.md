# Slip 2 – Q1: HashMap with entrySet()

## What the program does
Creates a `HashMap<Integer, String>` mapping roll numbers to student names,
inserts 5 entries, and displays all key-value pairs using `entrySet()`.

## Java Concepts Used
- **HashMap** – key-value store with O(1) average lookup
- **entrySet()** – returns a `Set<Map.Entry<K,V>>` for iteration
- **Generics** – `HashMap<Integer, String>`
