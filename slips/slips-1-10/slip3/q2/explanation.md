# Slip 3 – Q2: Abstract Class Person → Employee

## What the program does
Defines an **abstract class** `Person` with an abstract method `showDetails()`.
The concrete subclass `Employee` extends `Person` and provides the implementation.
Two employee objects are created and their details displayed.

## Java Concepts Used
- **Abstract Class** – `abstract class Person` cannot be instantiated
- **Abstract Method** – `abstract void showDetails()` forces subclass to implement
- **Inheritance** – `Employee extends Person`
- **super()** – calls parent constructor
