# Slip 3 – Q1: HashSet with Duplicates

## What the program does
Stores names in a `HashSet`, intentionally adds duplicate values, then
displays the set to observe that **duplicates are automatically rejected**.

## Java Concepts Used
- **HashSet** – unordered Set that rejects duplicates using `hashCode()` + `equals()`
- **Set interface** – contract that guarantees uniqueness
- **Hashing** – internally used to store elements
