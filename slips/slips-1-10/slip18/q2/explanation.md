# Slip 18 тАУ Q2: Method Overloading тАУ Calculator add()

## What the program does
Defines `Calculator` with **4 overloaded `add()` methods** covering `int`, `double`,
`long`, and 3-argument versions. Java selects the correct one at compile time.

## Java Concepts Used
- **Method Overloading** тАУ same name, different parameter types/counts
- **Compile-time Polymorphism** тАУ selection at compile time
- Types: `int+int`, `double+double`, `int+int+int`, `long+long`
