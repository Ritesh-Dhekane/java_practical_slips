# Slip 18 – Q1: Servlet – Display Date & Time

## What the program does
A servlet mapped to `/datetime` that responds to GET requests by generating
an HTML page showing the **current server date and time** using `new Date()`.

## Java Concepts Used
- **Servlet** – `extends HttpServlet`
- **doGet()** – handles HTTP GET requests
- **java.util.Date** – retrieves current system date and time
- **PrintWriter** – writes HTML to browser
