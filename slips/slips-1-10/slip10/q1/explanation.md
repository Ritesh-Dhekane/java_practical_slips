# Slip 10 – Q1: Employee Encapsulation

## What the program does
Defines an `Employee` class with **all fields declared `private`**.
Provides `public` getter and setter methods with validation.
Creates employee objects using both constructor and setters.
Displays employee details exclusively through getter methods.

## Java Concepts Used
- **Encapsulation** – `private` fields, `public` getters/setters
- **Data Hiding** – direct field access prevented from outside
- **Validation in Setters** – salary cannot be negative, ID must be positive
- **Constructor** – both default and parameterized
- **Access Modifiers** – `private`, `public`
