# Slip 1 – Q2: Lambda Expression for x^y

## What the program does
Defines a custom functional interface `Power` and uses a **Lambda Expression**
to implement it. Computes x raised to the power y using `Math.pow()`.

## Java Concepts Used
- **Lambda Expression** – concise anonymous function syntax `(x, y) -> Math.pow(x, y)`
- **Functional Interface** – `@FunctionalInterface` with single abstract method
- **Math.pow()** – standard library power function
