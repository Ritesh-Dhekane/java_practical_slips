# Slip 17 – Q1: Vector – Employee Search

## What the program does
Stores 5 employee names in a `Vector`. Accepts a name from the user and uses
`Vector.contains()` to check membership. Displays the index if found.

## Java Concepts Used
- **Vector** – synchronized dynamic array (thread-safe alternative to ArrayList)
- **contains()** – checks if element exists
- **indexOf()** – returns position of element
