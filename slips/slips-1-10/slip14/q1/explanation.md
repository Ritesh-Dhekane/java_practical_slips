# Slip 14 – Q1: LinkedList – Remove User Element

## What the program does
Creates a `LinkedList<String>` with 5 city names. Accepts a city name from the user
and removes it using `LinkedList.remove(Object)`. Displays the remaining list with size.

## Java Concepts Used
- **LinkedList** – doubly-linked list; efficient insertion/deletion
- **remove(Object)** – removes first occurrence; returns `true` if found
- **Scanner** – reads user input
