# Slip 14 – Q2: Constructor Overloading in Person

## What the program does
Defines `Person` with **four overloaded constructors** — no-arg, name-only,
name+age, and name+age+city. Each object is created using a different constructor,
demonstrating how Java selects the right one by parameter signature.

## Java Concepts Used
- **Constructor Overloading** – multiple constructors with different parameter lists
- **this.field** – differentiates instance fields from parameters
- **Object creation** – `new Person(...)` triggers the matching constructor
