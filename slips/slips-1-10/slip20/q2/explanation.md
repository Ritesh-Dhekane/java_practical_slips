# Slip 20 – Q2: Reverse a Sentence

## What the program does
Accepts a full sentence from the user and reverses the **entire character sequence**
(not word-by-word) using `StringBuilder.reverse()`. Displays original and reversed.

## Java Concepts Used
- **StringBuilder** – mutable string with `reverse()` method
- **Scanner.nextLine()** – reads full line including spaces

## Syllabus Unit
**UNIT 1** – Class & Objects (String manipulation)
