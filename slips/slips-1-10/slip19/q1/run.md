# How to Run – Slip 19 Q1 (JSP)

## Prerequisites
- Apache Tomcat 9.x

## Project Structure
```
CustomerApp/
├── index.html
├── display.jsp
└── WEB-INF/web.xml
```

## Deploy & Run
1. Copy `CustomerApp/` to Tomcat `webapps/`
2. Start Tomcat
3. Open: `http://localhost:8080/CustomerApp/index.html`
4. Enter name and city, click Submit
