# Slip 12 – Q1: TreeMap – Sorted by Key

## What the program does
Creates a `TreeMap<Integer, String>` mapping roll numbers to names,
inserts 5 entries in random order, and displays them. `TreeMap` **automatically
sorts by key** in ascending order.

## Java Concepts Used
- **TreeMap** – sorted Map implementation (Red-Black Tree internally)
- **entrySet()** – iterate over key-value pairs
- **firstKey() / lastKey()** – utility methods
