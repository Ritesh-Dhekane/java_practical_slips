# Slip 11 – Q1: ArrayList – Department Search

## What the program does
Stores 5 department names in an `ArrayList`. Accepts a name from the user
and checks if it exists using `ArrayList.contains()`. Displays an appropriate message.

## Java Concepts Used
- **ArrayList** – stores department names
- **contains()** – checks membership in O(n)
- **Scanner** – reads user input

## Syllabus Unit
**UNIT 2** – Collections, ArrayList
