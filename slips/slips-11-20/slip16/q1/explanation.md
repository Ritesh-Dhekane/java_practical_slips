# Slip 16 – Q1: Student – Total & Percentage

## What the program does
Creates a `Student` class with marks for 3 subjects. Calculates and displays
`total`, `percentage`, and `grade` for 3 student objects.

## Java Concepts Used
- **Class & Objects** – `Student` with fields and methods
- **Methods** – `getTotal()`, `getPercentage()`, `getGrade()`, `display()`
- **Array of objects** – `Student[]`

## Syllabus Unit
**UNIT 1** – Class & Objects
