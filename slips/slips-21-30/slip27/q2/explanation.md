# Slip 27 – Q2: JSP Directives
## What the program does
Explains and demonstrates all 3 JSP directives:
1. **page** – sets language, encoding, imports, error handling
2. **include** – statically includes another file at translation time
3. **taglib** – declares JSTL/custom tag libraries
