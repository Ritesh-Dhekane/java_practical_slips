# Slip 21 – Q2: JSP Scripting Elements
## What the program does
Demonstrates all 3 JSP scripting elements:
1. **Scriptlet** `<% %>` – Java code that runs on each request
2. **Expression** `<%= %>` – evaluates and outputs value
3. **Declaration** `<%! %>` – declares class-level methods/variables
