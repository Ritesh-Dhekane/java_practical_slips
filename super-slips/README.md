# Super Slips

Super Slips are reference implementations, not exam slips.

They exist to provide:
- complete learning examples
- reusable project structures
- gold-standard workflow references
- implementation templates for recurring concepts

Available reference projects:
- `servlet-super-slip/`
- `jsp-super-slip/`
- `jdbc-super-slip/`
- `multithreading-super-slip/`
- `collections-super-slip/`

These projects are intended to complement the normal `slips/` folder. Future slip fixes and improvements can refer back to these examples instead of recreating the same infrastructure repeatedly.
